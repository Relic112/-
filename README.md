## 一、界面

如图共三栏，左上角为输入框，左下角为输出框，右边为历史记录。

## 二、功能

可输入包含 ‘+’， ‘-’， ‘*’， ‘/’ ，‘^’共五个基本运算符及数字括号进行计算。

## 三、使用方法：
1.输入表达式，按下回车键即可输出。
2.按下键盘上键将会将输出结果放到输入框。
3.按下键盘下键清空输入框。
